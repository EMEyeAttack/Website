# EM Eye Website

This repository hosts the project website for **EM Eye: Characterizing Electromagnetic Side-channel Eavesdropping on Embedded Cameras**, published at NDSS 2024.

EM Eye pioneered the systematic study of content-level electromagnetic side-channel eavesdropping on embedded camera and image-sensor data transmission, laying groundwork for the broader research line on **sensor electromagnetic (EM) side-channel leakage**.

- Project website: https://emeyeattack.github.io/Website/
- Official NDSS page: https://www.ndss-symposium.org/ndss-paper/em-eye-characterizing-electromagnetic-side-channel-eavesdropping-on-embedded-cameras/
- Official paper PDF: https://www.ndss-symposium.org/wp-content/uploads/2024-552-paper.pdf
- Code and tutorial: https://github.com/longyan97/EMEye_Tutorial
- Research-lineage page: https://emeyeattack.github.io/Website/sensor-em-side-channel-lineage.html
